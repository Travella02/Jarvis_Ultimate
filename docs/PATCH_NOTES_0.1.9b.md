# Jarvis Ultimate 0.1.9b — Holographic Interface and Sleep Mode Hotfix

This hotfix refines the native app shell visuals after the 0.1.9a interface control update.

## Changes

- Fixed the voice panel header/buttons overflowing or hiding inside the left rail.
- Made sleep/wake waiting mode a constant dim grey visual state instead of shifting between grey-blue tones.
- Added registered state color channels so the orb and panel colors blend/fade between modes instead of snapping instantly.
- Darkened the full UI background to an almost pitch-black base.
- Made panels more holographic, blue-tinted, and partially transparent.
- Kept Auto Wake startup behavior from 0.1.9a.
- Added tests for the new holographic sleep mode and blended state transition behavior.

## Version

App shell bridge version is now `0.1.9b`.
