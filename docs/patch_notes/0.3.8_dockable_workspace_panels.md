# Jarvis Ultimate 0.3.8 — Dockable Workspace Panels

## Summary

This update turns Jarvis's app shell into a more flexible workspace. Panels can be arranged with the mouse, resized, saved, restored, minimized, popped out, and opened from a small panel command input.

## Changes

- Adds mouse-draggable panels.
- Adds mouse-resizable panels.
- Saves panel positions, sizes, minimized state, and floating/docked state to local storage.
- Restores the user's previous layout on startup.
- Adds layout lock/unlock mode.
- Adds reset layout control.
- Adds saved layout presets: Gaming, Coding, Music, and Minimal.
- Adds panel command input for quickly opening panels by name.
- Adds panel popout windows so panels can be moved to another monitor.
- Keeps existing panel show/hide toggles and orb-only mode.
- Adds capability flags and tests for the new UI workspace behavior.

## Notes

Panel popouts are a first foundation. They are intended to be useful for moving a panel to another monitor, but future updates can make popped-out panels fully interactive and synchronized through a dedicated Electron window manager.
