# Jarvis Ultimate 0.3.7a — Memory Review Cleanup Hotfix

## Summary

This hotfix cleans up the new memory review panel output before moving on to the larger UI panel work.

## Changes

- Hides internal memory/source labels such as `attribute_backfill` from user-facing memory review bullets.
- Deduplicates repeated relationship bullets such as duplicate fiancée records.
- Normalizes fiancé/fiancée display in memory review items.
- Keeps sensitive redaction active for review output.
- Adds tests for memory review cleanup behavior.

## Notes

This is a display cleanup only. It does not remove valid memories; it prevents internal bookkeeping labels from appearing in the panel.
